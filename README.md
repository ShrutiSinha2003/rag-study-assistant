# 📚 AI Study Assistant (RAG)

An AI-powered Retrieval-Augmented Generation (RAG) application that allows users to upload PDFs and ask contextual questions from documents.

## Features
- PDF Upload
- Text Chunking
- Vector Embeddings
- Semantic Search
- AI-generated Answers
- Streamlit UI

## Tech Stack
- Python
- Streamlit
- ChromaDB
- Scikit-learn
- Groq API
- TF-IDF Vectorization

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
